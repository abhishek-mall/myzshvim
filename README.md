A minimalist, high-performance setup for macOS (iTerm2).

## Features
- **Shell:** Zsh with [Powerlevel10k](https://github.com/romkatv/powerlevel10k) (Rainbow/Round style).
- **Editor:** Vim with `vim-plug`, customized for speed and aesthetics.
- **Terminal:** Optimized for iTerm2 with Nerd Fonts.
- **Tools:**
  - `fzf` for fuzzy finding history and files.
  - `eza` as a modern replacement for `ls`.
  - `bat` as a modern replacement for `cat`.
  - Syntax Highlighting & Autosuggestions.

## Prerequisites
1. **iTerm2** installed.
2. **Font:** [MesloLGS NF](https://github.com/romkatv/powerlevel10k-media/raw/master/MesloLGS%20NF%20Regular.ttf) (required for icons).
3. **Packages:**
